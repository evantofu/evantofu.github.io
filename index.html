EV Market Intelligence Platform

Data science pipeline analyzing electric vehicle performance, market dynamics, and infrastructure gaps through advanced statistical modeling and machine learning techniques

  • K-means Clustering Analysis with elbow method optimization to identify distinct market segments and consumer categories
  • Time Series Forecasting using statistical models to predict sales trends with seasonal decomposition and confidence intervals
  • Correlation Matrix Analysis revealing efficiency-range trade-offs and battery capacity relationships across manufacturers
  • Geospatial Visualization of 19,915+ charging stations with network distribution analysis and urban/rural coverage gaps
  • Interactive Streamlit Dashboard with Plotly visualizations enabling real-time filtering and multi-dimensional exploration

LockIn Study Wellness Monitor

AI-powered study companion using computer vision and behavioral analytics to prevent burnout through real-time posture and stress monitoring with personalized interventions.

  • Computer Vision Integration with Google Gemini API for real-time posture analysis and facial expression detection from webcam feeds
  • WebRTC Frame Capture with OpenCV processing in Node.js backend for efficient image analysis without specialized hardware
  • Real-time Behavioral Analytics using Firebase Firestore to track posture lapses, stress peaks, and study session patterns
  • Automated Intervention System with browser-based Text-to-Speech API delivering personalized audio alerts and focus reminders
  • Session Analytics Dashboard generating downloadable PDF reports with jsPDF for comprehensive study wellness insights
